SOFTWARE LIBRE PARA TODOS - DOCUMENTACIÓN

Hecho por: Rody
Materia: Tecnología WEB

Esta web está diseñada para escuelas públicas con computadoras viejas.

Funciona sin internet rápido:
- No usa videos embebidos (solo miniaturas)
- Todas las imágenes son ligeras
- Solo HTML, CSS y JavaScript básico

Archivos:
- index.html → página principal
- style.css → diseño accesible
- script.js → menú interactivo
- imagen-inicio.jpg → foto de apoyo

Instrucciones:
1. Abrir index.html en cualquier navegador
2. Hacer clic en los enlaces para acceder a tutoriales y descargas
3. ¡No requiere instalación!

¡Gracias por ayudar a democratizar la tecnología!
